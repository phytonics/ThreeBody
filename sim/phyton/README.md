A physics library based on Python.


## A Dynamics Physics Library
```python
>>> from phyton import *
>>> distance = m[5]
>>> displacement = m[5, 5, 5]
>>> displacement
(5.0, 5.0, 5.0) m
>>> distance
5.0 m
>>> displacement.mag
8.660254037844387 m
>>> print(displacement.unit)
m
>>> print(area.unit)
m^2
>>> length = m[5]
>>> width = m[3]
>>> area = length * width
>>> area
15.0 m^2
```
